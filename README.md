# 데이터 시각화 커리큘럼

[Vega-Lite](https://vega.github.io/vega-lite/)와 [Altair](https://altair-viz.github.io/)를 사용하는 대화형 노트북으로 구성된 데이터 시각화 커리큘럼입니다. 이 저장소에는 **Python** 기반의 일련의 Jupyter 노트북이 포함되어 있습니다. 노트북은 [Jupyter book](https://uwdata.github.io/visualization-curriculum/intro.html)에서 온라인으로 볼 수 있으며, 로컬에서 실행하거나 [Colab](https://colab.research.google.com/github/uwdata/visualization-curriculum), [Nextjournal](https://github.nextjournal.com/uwdata/visualization-curriculum), 또는 [Deepnote](https://deepnote.com/launch?url=https://github.com/uwdata/visualization-curriculum/blob/master/altair_introduction.ipynb)에서 온라인으로 실행할 수 있습니다. 해당되는 **JavaScript** 노트북 세트는 [Observable](https://observablehq.com/@uwdata/data-visualization-curriculum)에서 온라인으로 이용 가능합니다.

## 시작하기

시각화 커리큘럼은 온라인이나 로컬 컴퓨터에서 사용할 수 있습니다.

### 온라인

- [Jupyter Book](https://uwdata.github.io/visualization-curriculum/)에서 온라인으로 Python 노트북을 읽을 수 있습니다.

### 로컬 설치

1. [`uv` 설치](https://docs.astral.sh/uv/getting-started/installation/).
2. 저장소를 복제하고 의존성을 설치합니다:
   ```bash
   git clone https://github.com/uwdata/visualization-curriculum.git
   cd visualization-curriculum
   uv sync
   ```
3. 로컬 노트북 환경에서 노트북을 엽니다. 예:
   ```bash
   uv run jupyter lab
   ```

프로그래밍 환경(및 실시간 인터넷 연결 여부)에 따라 Altair를 위한 특정 [렌더러](https://altair-viz.github.io/user_guide/display_frontends.html)를 지정해야 할 수도 있습니다.
